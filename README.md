# MadDrive team Robofest signs detector

## Требования

- Python3.8

## Описание

Детектор знаков и светофора, используемый на соревнованиях Autonet/Robofest. Данный детектор позволяет определить знаки "Движение прямо", "Движение направо", "Движение налево", "Движение прямо или направо", "Движение прямо или налево", "Въезд запрещен", а также зону светофора на изображении. Использование предполагает установку на малую беспилотную платформу в масштабе 1:8 от реального автомобиля. Обучение нейросети проводилось на данных с самих соревнований, поэтому применение детектора на данных на реальной дороги может иметь меньшую точность. Применение детектора в рамках соревнований позволяет решать вопрос определения правильного анправления движения с учетом расставленных знаков по правилам соревнований.
